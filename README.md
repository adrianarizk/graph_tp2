# Graph Database – TP2 (S5)

This project demonstrates an end-to-end data pipeline connecting a relational database (**PostgreSQL**) to a graph database (**Neo4j**) using **Python, FastAPI, and Docker**.

It was developed as part of the course **Graph Databases – S5** and showcases how to extract, transform, and visualize e-commerce data.
